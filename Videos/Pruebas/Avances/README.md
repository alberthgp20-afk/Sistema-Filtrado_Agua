# Avances

Esta carpeta contiene las evidencias del proceso de construcción e integración del prototipo.

Aquí se incluyen fotografías del ensamblaje de la estructura, instalación de los recipientes, organización del sistema de filtración, montaje del cableado y ubicación de los diferentes componentes electrónicos.

Estas evidencias muestran la evolución del proyecto antes de su versión final.
