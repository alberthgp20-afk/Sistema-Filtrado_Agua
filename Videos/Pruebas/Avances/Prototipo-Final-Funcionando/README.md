# Prototipo Final Funcionando

En esta carpeta se presentan las evidencias del prototipo completamente ensamblado y en funcionamiento.

Las imágenes muestran el sistema de tratamiento de agua terminado, incluyendo la estructura, el sistema de filtración, el ESP32, la pantalla de visualización, las bombas de agua y los sensores utilizados durante las pruebas finales.

Estas evidencias corresponden a la versión entregada del proyecto.
