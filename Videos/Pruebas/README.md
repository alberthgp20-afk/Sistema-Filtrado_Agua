# Pruebas

En esta carpeta se encuentran las evidencias correspondientes a las pruebas individuales realizadas durante el desarrollo del proyecto.

Estas pruebas permitieron verificar el correcto funcionamiento de los sensores, bombas de agua, pantalla, ESP32 y demás componentes antes de integrarlos al prototipo final.

Las imágenes muestran el proceso de calibración, conexión y validación de cada uno de los elementos utilizados.
